# 🏋️ AI Fitness Coach

An AI-assisted fitness tracking web app that helps users log weight, track progress, and eventually receive personalized fitness insights.

This project is built as a full-stack application using **Next.js (frontend)** and **FastAPI (backend)**.

---

# 🚀 Current Status

### ✅ Completed
- FastAPI backend setup
- Next.js frontend setup
- Basic project structure
- Health check API (`/health`)
- Development environment working

### 🚧 In Progress

- Frontend ↔ backend connection

### ⏳ Planned

- Smart recommendations
